Pokiaľ máš záujem hrať hru za STU, musím ti pridať rolu pre danú hru.
Hru si možeš zvoliť nasledovne pomocou týchto príkazov:

***League of Legends***
```
!lol [nick#server]
Napríklad -> !lol Ramang#eune, !lol lord ramang#euw.
```
Keď ti bude pridelená rola "League of Legends", môžeš použiť príkaz a bude ti pridelená aj rola akú hrávaš:
```
!lolrole [TOP,JUNGLE,MID,ADC,SUPP]
Napríklad -> !lolrole ADC
```  

***Dota 2***
```
!dota [steamid], zadaj svoje steam id.
Napríklad -> !dota 76561198058614248
```

***CS:GO***
```
!csgo [faceitid], zadaj svoje faceit meno (id) .
Napríklad -> !csgo DIVIX
```

***Valorant***
```
!valorant [name#tagline]
Napríklad -> !valorant ramang#eune
```


Ak nemáš faceit a máš záujem o pridanie CSGO role prosím kontaktuj, jedného z adminov discordu aby ti pridelil rolu.
